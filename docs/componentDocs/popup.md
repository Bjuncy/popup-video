# popup
弹窗组件

### 示例
<p-demo />

### 代码

``` javascript
popup({
    width: '500px',
    height: '300px',
    title: 'popup组件'
})
```

### Attributes

| 参数 | 说明 | 类型 | 是否必要 | 默认值 |
| ---  | ---  | ---  |   ---   |  ---  |
| width | 弹窗宽度 | String | false | 100% |
| haight | 弹窗高度 | String | false | 100% |
| title | 弹窗标题 | String | false |  |
| pos | 弹窗位置 | String | false | center |
| mask | 弹窗mask | String | false | true |
| content | 弹窗内容 | Element | false |  |
